**생성된 repository 를 로컬에 Checkout** 

Sync 하려는 로컬 경로로 이동하고, Git Bash 를 이용해서 Checkout

```
> git init
> git clone [repository 경로]
> git pull
```



**로컬에서 파일 수정 후 repository 업로드** 

```
> git add .
> git commit -m "커밋 로그"
> git push -u origin master
```
